# About

This repository is project skeleton for Go CLI application.

# Requirements

* go 1.18+
* gotestsum
* gomock

# Usage

```
git clone --depth 1 ssh://git@github.com/t-kuni/go-cli-app-skeleton [ProjectName]
cd [ProjectName]
rm -rf .git 
```

# Build

```
make build
```

# Run

```
go run ./main.go command1
```

# Run test

```
make test
```
